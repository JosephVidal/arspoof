# Arspoof
Spoofing arp script in python2.7 using scapy.

# Usage
```
$ ./arspoof.py -h
USAGE:
	./arspoof.py <ip_target> <ip_victim>
```
Install:
```
$ make install
```
Uninstall:
```
$ make uninstall
```
# Nota
The script must be run twice, one for target and one other for the router.
